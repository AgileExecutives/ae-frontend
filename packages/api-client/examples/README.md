# Vue API Client Examples

This directory contains examples of how to use the AE SaaS API Client in Vue applications.

## Files

- `setup.ts` - How to set up the client in your Vue app
- `LoginComponent.vue` - Authentication example
- `CustomerList.vue` - Resource management with CRUD operations
- `SearchComponent.vue` - Search functionality
- `main.ts` - Complete app setup example

## Usage

1. Install the package in your Vue project:
```bash
npm install @unburdy/ae-saas-api-client
```

2. Set up the client in your main.ts file (see setup.ts example)

3. Use the composables in your components (see component examples)
